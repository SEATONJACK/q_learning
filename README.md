## 更新函數

$$Q(s(n+1), a(n+1)) \leftarrow Q(s(n), a(n)) + \alpha[r(n+1) + \gamma \max \limits_{a} Q(s(n+1))-Q(s(n),a(n))]$$

##  $\epsilon$-greedy策略

* (1-$\epsilon$)的機率採用動作$a=arg\max\limits_{a}Q_{t}(s, a)$
* $\epsilon$的機率**隨機**採用其他action
